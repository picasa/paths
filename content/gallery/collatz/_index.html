Curves parameterized from integer sequences and positioned according to basic botanical rules.

This serie focus on using a very simple generator, here a sequence of numbers. A sequence of integers is generated from a set of rules (here, the Collatz sequence). Then, a curve is defined by mapping sequence elements to segment length, with a fixed angle between segments. Multiple curves are computed to constitute a node, and multiple nodes to constitute a stem. The only random elements are the initial value of the sequence, and the angle between curve segments.
